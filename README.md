# Simple Calculator

## Overview

This is a simple web-based calculator built using **HTML, CSS, and JavaScript**. It performs basic arithmetic operations with a clean and responsive user interface inspired by modern smartphone calculators.

## Features

* Addition (+)
* Subtraction (−)
* Multiplication (×)
* Division (÷)
* Percentage (%)
* Decimal number support
* Backspace (⌫)
* All Clear (AC)
* Responsive and user-friendly design

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

## How to Run

1. Download or clone the project.
2. Open the `calculator.html` file in any modern web browser.
3. Start performing calculations.

## How to Use

1. Click the number buttons to enter values.
2. Select an arithmetic operator (+, −, ×, ÷).
3. Enter the second number.
4. Click the **=** button to display the result.
5. Use **AC** to clear the display.
6. Use **⌫** to delete the last entered character.
